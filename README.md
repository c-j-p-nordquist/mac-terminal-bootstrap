# MacOS Terminal Setup Script

This repository contains a bash script for setting up a new MacOS Terminal environment. The script installs Homebrew, Zsh, Oh My Zsh, AWS CLI, and several other useful terminal applications. It also sets up a couple of useful Oh My Zsh plugins.

## Usage

1. Clone this repository to your local machine:
    ```bash
    git clone https://github.com/c-j-p-nordquist/dev-env-bootstrap.git
    ```

2. Navigate to the repository directory:
    ```bash
    cd dev-env-bootstrap
    ```

3. Make the script executable:
    ```bash
    chmod +x bootstrap_mac_arm.sh
    ```

4. Run the script:
    ```bash
    ./bootstrap_mac_arm.sh
    ```
